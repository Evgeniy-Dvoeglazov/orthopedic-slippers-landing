# Orthopedic slippers
* Одностраничный лендинг для магазина ортопедической обуви

## Функционал проекта

* Свайпер из нескольких элементов с превью
* Ховер при наведении на кнопки
* Таймер обратного отсчета
* Кастомные селекты

## Технологии

* HTML
* CSS
* JavaScript
* Сhoices.js
* Swiper.js
